# TruthLens AI

Explainable Fake News Detection using Retrieval-Augmented Generation (RAG), BERT, Sentence Transformers, and FAISS.

## Features

- Fine-tuned BERT classifier
- Retrieval-Augmented Generation (RAG)
- Semantic search using Sentence Transformers
- FAISS vector database
- Explainable evidence retrieval
- Modern React + Vite frontend
- FastAPI backend
- PDF report export
- Recent analysis history

## Tech Stack

- React
- Vite
- Tailwind CSS
- FastAPI
- PyTorch
- Transformers
- Sentence Transformers
- FAISS

## Run Backend

cd backend

pip install -r requirements.txt

uvicorn app:app --reload

## Run Frontend

cd frontend

npm install

npm run dev
